<a id="chapter-1"></a>
## Installation


### Install from source:
```bash
git clone https://github.com/thek4n/note.git
cd note-script
make install
```

### Install by pacman (Recommended):
```bash
git clone https://github.com/thek4n/note.git
cd note
makepkg -sic
```


<a id="chapter-2"></a>
## Usage

```bash
note init
note edit someNote.md
note render so<Tab>  # open localhost:6751 in browser
```
